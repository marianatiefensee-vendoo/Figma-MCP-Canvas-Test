# Agents

## Figma MCP Routing

When a task requires Figma write capabilities, use the MCP tool that provides write access directly and do not keep searching between connections.

- If the task needs `generate_figma_design`, always use the remote Figma MCP server at `https://mcp.figma.com/mcp`.
- Do not route this capability through the desktop Figma server.
- Prefer the remote server whenever write access is required for Figma-related work.
